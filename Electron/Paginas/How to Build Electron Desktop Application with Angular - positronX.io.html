Cómo construir una aplicación de escritorio Electron con Angular

Ultima actualización en:10 de abril de 2022por Digamber
En este tutorial, vamos a aprender paso a paso cómo crear una aplicación de escritorio Electron básica multiplataforma con el marco frontal Angular.

Electron.js es una conocida plataforma de código abierto desarrollada por GitHub bajo la licencia MIT. Ayuda a crear una aplicación de escritorio multiplataforma robusta para varios dispositivos como macOS, Linus y Windows utilizando HTML, JavaScript y CSS. Para saber más sobre Electron.js puedes visitar su sitio oficial aquí .

Electron facilita la vida de los desarrolladores al administrar el trabajo duro por sí mismo. Utiliza Google Chromium y Node.js y le permite crear aplicaciones flexibles y potentes utilizando HTML, CSS y JS. Electron también ofrece su propia colección de API para crear aplicaciones web.

En este tutorial, veremos cómo configurar una aplicación Angular desde cero y ejecutarla dentro de Electron con la ayuda de TypeScript y su propio conjunto de API nativas.

Comencemos a construir Electron y Angular .

Requisito previo

Para comprender este tutorial a fondo, debe tener en cuenta los siguientes requisitos previos:

  • Debe tener experiencia con HTML, CSS, JavaScript y TypeScript.
  • Node.js y NPM instalados en su sistema, si no, siga este tutorial para configurar el entorno de desarrollo de Node JS

Los proyectos de Angular se desarrollan usando Angular CLI, es una herramienta oficial para trabajar con Angular. Ejecute el comando para instalar Angular CLI, ignore este paso si Angular CLI ya está instalado.

npm install @angular/cli -g

Instale la última versión de Electron en Angular 8 Project

En este paso, instalaremos la última versión de Electron dentro de la carpeta raíz del proyecto Angular 8.

Ejecute el siguiente comando para configurar Electron en el proyecto Angular:

npm install electron@latest --save-dev
Intento

Como puede ver, estamos instalando el paquete Electron como una dependencia de desarrollo porque queremos usar este paquete durante la fase de desarrollo. No queremos que este paquete se incluya en la aplicación principal.

Una vez que el electrón se haya instalado con éxito, puede verificar la versión del electrón ejecutando el siguiente comando en la terminal:

electron --version
# v6.0.1
Intento

Estamos usando Electron v6.0.1 al momento de crear este tutorial.

Inicializar la aplicación Electron

Para configurar electron dentro del proyecto Angular, cree un nuevo archivo en la raíz del proyecto Angular. Nómbrelo app.jsy guarde todo el código relacionado con Electron dentro de él.

Ejecute el comando para crear un app.jsarchivo:

touch app.js

Comience a construir la aplicación Electron

Vaya al app.jsarchivo y agregue las dependencias requeridas para inicializar la aplicación Electron en Angular 8:

const url = require("url");
const path = require("path");

Defina una variable que se referirá a appWindow, y Esta es la misma ventana que Electron abrirá para alojar nuestra aplicación Angular 8.

Configuración de la ventana electrónica

Declare initWindowla función, y esta función creará una nueva ventana de electrones que ayudará a Angular a alojar nuestra aplicación de electrones.

Al usar el BrowserWindowmétodo, declaramos las dimensiones de la ventana para la ventana de electrones.

function initWindow() {
  appWindow = new BrowserWindow({
    width: 1000,
    height: 800,
    webPreferences: {
      nodeIntegration: true
    }
  })
}
JavaScript

También puede declarar la propiedad de pantalla completa dentro del método BrowserWindow para mostrar la ventana de la aplicación de electrones en pantalla completa.

function initWindow() {
  appWindow = new BrowserWindow({
    fullscreen: true
  })
}

Configurar ruta de compilación de electrones

Esta función definirá la ruta de compilación de la aplicación Electron.

function initWindow() {
  appWindow.loadURL(

      protocol: "file:",
      slashes: true
    })
  );
}

Ahora registre los oyentes de eventos en el objeto de electrones, estos oyentes inicializan la aplicación de electrones.

JavaScript

Archivo app.js final

const {
  app,
  BrowserWindow
} = require('electron')
const url = require("url");
const path = require("path");
let appWindow
function initWindow() {
  appWindow = new BrowserWindow({
    width: 1000,
    height: 800,
    webPreferences: {
      nodeIntegration: true
    }
  })
  // Electron Build Path
  appWindow.loadURL(
    url.format({
      pathname: path.join(__dirname, `/dist/index.html`),
      protocol: "file:",
      slashes: true
    })
  );
  // Initialize the DevTools.
  appWindow.webContents.openDevTools()
  appWindow.on('closed', function () {
    appWindow = null
  })
}
app.on('ready', initWindow)
// Close when all windows are closed.
app.on('window-all-closed', function () {
  // On macOS specific close process
  if (process.platform !== 'darwin') {
    app.quit()
  }
})
app.on('activate', function () {
  if (win === null) {
    initWindow()
  }
})
JavaScript

Manejar Electron no pudo cargar errores de secuencia de comandos del módulo

Si obtiene los siguientes errores cuando se inicia la ventana Electron:

  • Error al cargar la secuencia de comandos del módulo: el servidor respondió con un tipo MIME que no es de JavaScript de "". Se aplica una estricta verificación de tipo MIME para los scripts de módulo según la especificación HTML.
